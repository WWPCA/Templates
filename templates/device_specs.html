Device Requirements

About Device Requirements

IELTS AI Prep is designed to work on a wide range of devices. However, certain features, especially the speaking assessment, require specific capabilities. This page explains the minimum requirements and offers solutions for optimal performance.

Minimum Device Requirements

Computer / Laptop

  • Windows 10 or higher / macOS 10.13 or higher
  • 4GB RAM (8GB recommended)
  • Chrome, Firefox, Edge, or Safari (latest versions)
  • Webcam and microphone for speaking assessment
  • Stable internet connection (2 Mbps or higher)

Mobile Device

  • iOS 13 or higher / Android 9.0 or higher
  • 2GB RAM (4GB recommended)
  • Chrome or Safari (latest versions)
  • Working microphone for speaking assessment
  • Stable internet connection (1.5 Mbps or higher)

Feature-Specific Requirements

Low-Bandwidth Mode

IELTS AI Prep offers a low-bandwidth mode for users with limited internet connectivity. This mode:

  • Reduces image quality to minimize data usage
  • Optimizes data usage for slow connections
  • Prioritizes text content over multimedia
  • Optimizes speaking assessment audio quality

Low-bandwidth mode is automatically activated when a slow connection is detected, or you can enable it manually from the user settings panel.

Troubleshooting

  1. Check that your microphone is properly connected
  2. Ensure you've granted permission for the browser to access your microphone
  3. Try using a different browser (Chrome or Firefox recommended)
  4. If using mobile, ensure the app has microphone permissions
  5. Check if your microphone is working in other applications

  1. Check your device volume and ensure it's not muted
  2. Try using headphones to ensure you can hear properly
  3. Ensure your browser has permission to play audio
  4. Try refreshing the page or using a different browser
  5. Check if audio works in other applications or websites

  1. Enable low-bandwidth mode in your user settings
  2. Try connecting to a more stable internet connection
  3. Close other browser tabs and applications using the internet
  4. Clear your browser cache and cookies
  5. Try using a different device if available
